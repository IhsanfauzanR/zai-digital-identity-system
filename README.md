# ZAI Digital Identity System (ZDIS)

ZAI Digital Identity System is a personal web-based platform designed to represent a digital identity
through narrative, visual immersion, and interactive elements.

This project is built with a long-term vision, focusing on clarity of system design,
maintainability, and meaningful user experience.

## Project Status
- Documentation: In Progress (SRS & SDD completed)
- Backend: Planned
- Frontend: Planned

## Tech Stack
- Frontend: Vite + React, Tailwind CSS, Framer Motion
- Backend: Node.js, Express.js
- Database: PostgreSQL (Prisma ORM)
